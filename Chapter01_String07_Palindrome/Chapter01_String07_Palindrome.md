# Chapter01_String05_ReverseChars

## Question

Determine whether a string is a palindrome. (Just consider alphanumeric characters and ignore case.)

Example:

```
input: A man, a plan, a canal: Panama
output: true
```

```
input: applelppA
output: true
```

```
input: apple
output: false
```
