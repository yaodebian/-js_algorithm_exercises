# Chapter01_String04_CountAndSay

## Question

Count and Say

For example: input is a integer(1 ≤ n ≤ 30) and output is an integer.

| input | output | remark                                                      |
| ----- | ------ | ----------------------------------------------------------- |
| 1     | 1      |                                                             |
| 2     | 11     | last integer can be read: `one 1` => `11`                   |
| 3     | 21     | last integer can be read: `two 1` => `21`                   |
| 4     | 1211   | last integer can be read: `one 2, one 1` => `1211`          |
| 5     | 111221 | last integer can be read: `one 1, one 2, two 1` => `111221` |

Now you are given an integer (1 ≤ n ≤ 30). Please output the result.
